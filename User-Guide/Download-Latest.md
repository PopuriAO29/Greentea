### Download latest build or release

Latest release (i.e. build that was tested by the developer) can be found here https://github.com/GreenteaOS/Greentea/releases

But you may also acquire latest un-tested one (no guarantees at all) at https://ci.appveyor.com/project/PeyTy/tofita/build/artifacts and in the "Actions" tabs of repos (they require GitHub account)
